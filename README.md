# OS Concepts Visualization

Interactive, browser-based visualization of classic OS concepts.

## Features
- CPU Scheduling (FCFS, SJF, RR, Priority) with Gantt chart, timeline scrubbing, and live stats
- Producer–Consumer with buffer/semaphore-like visuals
- Banker’s Algorithm with interactive matrices and safety check

## Structure
- index.html, style.css
- js/state.js, js/algorithms.js, js/draw.js, js/animation.js, js/ui.js
- script.js (bootstrap)

## Run
Open index.html in a modern browser. No build step needed.
